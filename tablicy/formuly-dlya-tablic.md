# Формулы для таблиц

В таблицах Salebot теперь можно в ячейках использовать встроенные формулы:

<div><figure><img src="../.gitbook/assets/2025-05-14 17.31.48.jpg" alt=""><figcaption></figcaption></figure> <figure><img src="../.gitbook/assets/2025-05-14 17.31.43.jpg" alt=""><figcaption></figcaption></figure></div>

Чтобы воспользоваться формулой, пропишите в ячейке "=", далее добавьте (БЕЗ пробела) необходимую формулу в виде буквенного значения (например, СУММ) и выберите значения из других ячеек или пропишите значения вручную.

Основные символы:

1. `=` — знак равно; чтобы активировать формулы, его прописывают первым символом в ячейке;
2. `СУММ` — пример формулы (может быть любая иная из таблицы ниже);
3. `()` — в скобки закрывают значения, с которыми осуществляются математические действия;
4. **`,`** — для указания очередности ячеек, действий или значений в формуле;
5. **`:`** — для обозначения диапазона ячеек, чтобы не перечислять каждую ячейку через запятую.

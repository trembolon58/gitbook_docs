# Как создать услуги

Чтобы ваши клиенты могли к вам записываться, необходимо добавить различные услуги.&#x20;

Каждая услуга относится к определенной категории, поэтому для начала необходимо создать категорию услуг. После создания филиала вы увидите кнопку "Создать категорию".

<figure><img src="../../.gitbook/assets/template 781.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Без создания категории, вы не сможете создать услугу.&#x20;
{% endhint %}

## Создание категорий услуг

Кликните на кнопку "**Создать категорию**", расположенной в середине экрана, как показано выше. После нажатия откроется окно для добавления новой категории. В поле "Название категории" выберите наименование категории, удобное для группировки ваших услуг.&#x20;

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 11.56.35.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
Например, категория "Танцы" - услуга "Народные танцы", или категория "Йога" - услуга "Йога на гамаках" и так далее.
{% endhint %}

После заполнения поля "Название категории" нажмите на кнопку "Сохранить", тогда категория будет добавлена в разделе услуги.

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 12.02.56.png" alt="" width="563"><figcaption></figcaption></figure>

Вы можете добавить любое количество категорий услуг, для этого найдите кнопку справа экрана "Добавить категорию":

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 12.07.36.png" alt=""><figcaption></figcaption></figure>

После того как вы добавили необходимое количество категорий, можно приступать к созданию услуг:

<figure><img src="../../.gitbook/assets/template 782 (1).png" alt=""><figcaption></figcaption></figure>

## Создание услуг

Для создания услуги кликните по кнопке "Создать", которая расположена в правой части экрана:

Далее откроется модальное окошко со всеми настройками услуг:

<figure><img src="../../.gitbook/assets/template 782 (2).png" alt="" width="563"><figcaption></figcaption></figure>

1. **Название услуги**

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 12.36.29.png" alt="" width="563"><figcaption></figcaption></figure>

Название услуги отображается в боте (в том числе с ИИ) и виджете.&#x20;

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 12.41.12.png" alt="" width="563"><figcaption></figcaption></figure>

2. **Категория услуг**

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 12.42.47.png" alt="" width="563"><figcaption></figcaption></figure>

Поможет отнести создаваемую услугу к одной из категорий.&#x20;

3. **Вместимость**

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 12.45.00.png" alt="" width="563"><figcaption></figcaption></figure>

Количество человек, которые одновременно (то есть в одно и то же время) могут записаться / забронировать вашу услугу.&#x20;

{% hint style="info" %}
Подходит для групповых записей на услуги.
{% endhint %}

7. Период выполнения услуги

Чекбокс "Период выполнения услуги" - это временные интервалы, в которые будет доступна услуга для записи:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 13.08.52.png" alt="" width="375"><figcaption></figcaption></figure>

То есть, можно установить временные интервалы, когда ваши клиенты могут записаться к вам на услугу:

<figure><img src="../../.gitbook/assets/Запись-экрана-2025-03-20-в-13.11.40.gif" alt="" width="563"><figcaption></figcaption></figure>

Именно часы, установленные в настройке "Период выполнении услуг", будут доступны для записи в боте или виджете. В случае, если вы не установите данную настройку, то клиенты смогут записаться на услугу в любое рабочее время вашего сотрудника филиала.&#x20;

{% hint style="warning" %}
Внимание!&#x20;

Период выполнения услуг должен быть строго больше по времени, чем сама услуга.&#x20;

Иначе услуга не будет доступна для записи.&#x20;
{% endhint %}

{% hint style="success" %}
Для каждой услуги можно установить свои интервалы: так вы сможете, например, записывать на стрижку только в понедельник, среду и пятницу, а проводить обучения по стрижке во вторник и в пятницу только в определенные часы.&#x20;

Причем также вы сможете комбинировать занятия, поставив в один день, но в разные часы как групповые занятия, так и одиночные: например, создав две услуги "групповая йога" и "личная тренировка" и установив для каждой услуги свои временные интервалы для записи.&#x20;
{% endhint %}

8. Интервалы между онлайн-записями

В случае, если вместимость равна 1, то можно добавить интервалы после каждой записи клиента, например, если вы не указываете период выполнения услуг:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-20 в 13.40.02.png" alt="" width="563"><figcaption></figcaption></figure>

Это поможет установить отдых между клиентами для сотрудника. Например, если длительность услуги равна одному часу, а интервал между онлайн-записями равен 10 минутам, первый клиент записался в 9:00, то следующее свободное окошко для клиента будет доступно через 1 час 10 минут, то есть в 10 часов 10 минут.

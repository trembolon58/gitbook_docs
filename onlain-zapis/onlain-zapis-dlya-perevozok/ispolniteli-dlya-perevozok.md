# Исполнители для перевозок

## Как создать исполнителя для перевозок

После настройки вкладки с услугами перейдите во вкладку "Исполнители":

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 11.48.33.png" alt=""><figcaption></figcaption></figure>

Здесь необходимо создать исполнителя: для этого нажмите на одноименную кнопку, после чего откроется модальное окошко с добавлением исполнителя:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 11.49.30.png" alt=""><figcaption></figcaption></figure>

Выберите тип сотрудника:

1. Рейсовый автобус

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 11.50.04.png" alt=""><figcaption></figcaption></figure>

Обязательное поле — Название рейса:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 11.51.09.png" alt=""><figcaption></figcaption></figure>

Затем при необходимости пропишите описание рейса и добавьте изображение:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 11.53.46.png" alt=""><figcaption></figcaption></figure>

Теперь для исполнителя типа "Рейсовый автобус" можно назначить услуги:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 11.54.33.png" alt=""><figcaption></figcaption></figure>

Вы можете создать любое необходимое количество рейсовых автобусах в разделе "Исполнители".

При клике на исполнителя откроются настройки:

1. Основная информация — для указания информации об автобусе (автомобиле)
2. Услуги — для назначения услуг;
3. Рабочий график — для установки времени и даты поездок.

## Основная информация

В разделе "Основная информация" можно изменить / добавить:

1. Название рейса;
2. Описание рейса;
3. Марку автобуса;
4. Гаражный номер;
5. Госномер;
6. ИП (АТП);
7. ИНН;
8. Условия приобретения билетов;
9. Изображение рейса (автобуса / автомобиля).

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.16.53.png" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
ИП — индивидуальный предприниматель.

АТП — автотранспортное предприятие.

В поле ИП (АТП) можно указать название вашего ИП или АТП.
{% endhint %}

Далее нажмите "Сохранить", чтобы информация была сохранения в разделе:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.18.12.png" alt="" width="563"><figcaption></figcaption></figure>

## Назначить рейс

Чтобы назначить рейсу определенные маршруты, которые вы прописали в разделе "Услуги", перейдите во вторую вкладку настроек исполнителя:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.20.07.png" alt=""><figcaption></figcaption></figure>

Далее нажмите кнопку "Назначить услуги":

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.21.01.png" alt=""><figcaption></figcaption></figure>

Откроется окно, в котором нужно выбрать, какие рейсы будет выполнять данный автобус:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.22.20.png" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
Можно назначить все рейсы, одну из категорий или один из рейсов.
{% endhint %}

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.24.26.png" alt=""><figcaption></figcaption></figure>

Далее нажмите "Сохранить". Тогда данному исполнителю типа "Рейсовый автобус" будут назначены выбранные вами рейсы.

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.25.40.png" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
Чтобы изменить рейсы для транспорта, кликните по кнопке "Назначить услуги" и выберите необходимые рейсы.
{% endhint %}

## Рабочий график

Чтобы назначить график для перевозок перейдите в раздел "Рабочий график":

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.28.46.png" alt=""><figcaption></figcaption></figure>

Здесь вы увидите шаблон для заполнения графика, где можно выставить:

1. Расписание:&#x20;

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.34.26.png" alt="" width="563"><figcaption></figcaption></figure>

Можно выставить, по каким дням будет осуществлять перевозки данный автобус (автомобиль).

2. Дату, с которой должен быть выставлен график:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.49.30.png" alt="" width="563"><figcaption></figcaption></figure>

3. Время работы:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.50.21.png" alt=""><figcaption></figcaption></figure>

В поле со временем работы нажмите на "Редактировать", чтобы открылся правый бар для настройки графика рейса.

Нажмите "Добавит рейс", тогда добавится плашка для настройки времени:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.52.13.png" alt=""><figcaption></figcaption></figure>

Выберите тип рейса и время отправления (время прибытия расчитается автоматически):

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.55.14.png" alt="" width="563"><figcaption></figcaption></figure>

Далее выберите на какое количество недель нужно выставить график:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.57.18.png" alt="" width="549"><figcaption></figcaption></figure>

Затем нажмите "Заполнить и сохранить":

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.58.10.png" alt="" width="563"><figcaption></figcaption></figure>

После того, как вы нажали на кнопку, чтобы заполнить график, будет создан утвержденный график:

<figure><img src="../../.gitbook/assets/Снимок экрана 2025-07-03 в 12.58.55.png" alt=""><figcaption></figcaption></figure>

В утвержденном графике можно увидеть:

1. Время начала поездки;
2. Время окончания поездки;
3. Количество пассажиров, забронировавших проезд;
4. Можно выгрузить путевой лист с информацией о пассажирах и их количестве;

Также можно удалить рейс при необходимости.

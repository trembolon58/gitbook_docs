# Секции (блоки)

В разделе Контент в каждой секции добавлено поле ID секции:

<figure><img src="../../../.gitbook/assets/image (95).png" alt=""><figcaption></figcaption></figure>

{% hint style="warning" %}
ID секции задаётся самостоятельно! \
Использовать можно только латиницу, числа и нижнее подчеркивание вместо пробелов.
{% endhint %}

ID секции блока можно использовать для настройки навигации по странице (меню сайта) и в своих скриптах для обращения к нужному блоку.

Изображения можно загружать файлом или ссылкой на облачное хранилище.&#x20;

Видео загружается ссылкой на размещенное в YouTube или Vimeo видео.

#### Рекомендации для изображений.

## Основные типы секций (блоков)

Настройки Контента определяют тип блока:

* [Универсальный блок](/broken/pages/xyl8etc4OH0St6lV8bto): в данном блоке можно добавлять изображения, галерею, видео и т.п. Является визитной карточкой вашего сайта и помогает адаптировать вашу страницу под любой удобный формат отображения: экран компьютера, телефона, планшета.&#x20;
* [Обложка](/broken/pages/6F6YkiC6UrTpAwH9GNwn): блок для добавления изображение и видео, заголовок.  Доступны шаблоны для добавления обложки страницы.&#x20;
* [Заголовок и текс](/broken/pages/dW4g0qINOoCqxI8pfqOv)т: текстовый блок с заголовком.&#x20;
* [HTML](/broken/pages/dW4g0qINOoCqxI8pfqOv): блок позволяет добавить на страницу HTML и CSS коды
* [Форма заявки с кнопками на мессенджеры](/broken/pages/NerUbzyC1sCXVvSoTA2e): блок для добавления кнопок мессенджеров, вопросов, политики конфиденциальности.&#x20;
* [Изображение или видео](/broken/pages/dW4g0qINOoCqxI8pfqOv): блок для добавления картинки или видео
* [Кнопки](/broken/pages/S4na9bSPZM4yI29COt4q): блок для добавления кнопок с функцией Ссылка, Телефон, email, Оплата
* [Цитата](/broken/pages/dW4g0qINOoCqxI8pfqOv): блок для добавления цитаты
* [Список](/broken/pages/dW4g0qINOoCqxI8pfqOv): блок с возможностью вставить нумерованные и маркированные списки
* [Выпадающий текст](/broken/pages/dW4g0qINOoCqxI8pfqOv): блок для добавления выпадающего текста
* [Галерея](/broken/pages/dW4g0qINOoCqxI8pfqOv): блок для добавления галереи изображений.&#x20;
* [Таймер](/broken/pages/dW4g0qINOoCqxI8pfqOv): блок для добавления таймера обратного отсчета.&#x20;
* Аккордеон: блок с добавлением текстовой информации
* [Формы](/broken/pages/YOzidl4HYJaWmkPXnuYY): блок для сбора данных, добавления анкетирования на курс
* [Виджеты онлайн-записи](/broken/pages/nRFzu0WTeKaztnvBy4Vw#vidzhet-onlain-zapisi-na-sait): блок для записи клиента на ваши услуги или бронирования объектов (например, столики в рестораны)
* [Блок поп-ап](/broken/pages/hxO6OwzOdQPIjJgFHJXc): блок-всплывающее окно&#x20;

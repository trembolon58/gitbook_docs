# Интернет-магазин

Теперь вы можете создать полноценный интернет-магазин прямо внутри Сейлбот: новый тип сайта "Интернет-магазин" позволяет быстро запустить витрину с товарами, не прибегая к сторонним решениям. Вся настройка — в удобном визуальном интерфейсе, без необходимости в программировании.

<figure><img src="../../.gitbook/assets/Запись-экрана-2025-07-23-в-16.04.01-_online-video-cutter.com_.gif" alt=""><figcaption></figcaption></figure>

Вы сможете собрать полноценный магазин: добавить товары, фото, описание, цену — а покупатели смогут выбрать, положить в корзину и оплатить. Всё работает прямо на сайте. Без лишней интеграции. Просто и быстро.

# Email-рассылки

{% hint style="success" %}
Рассылки доступны только на тарифах "Бизнес" и "Инфобиз".&#x20;

Рассылки недоступны на бесплатном тарифе, а также при пробном трехдневном тестовом периоде, который активируется при создании проекта.&#x20;
{% endhint %}

{% hint style="warning" %}
Важно!&#x20;

1. Максимальное количество запланированных сообщений на одного клиента — 32 сообщения.
2. Максимальное количество переменных для сравнения в рассылках — 6 шт.&#x20;
{% endhint %}

**Невалидные адреса email** - адреса, которые написаны с ошибками в домене (@mail.ru) или префиксе (name@). Невалидными также называют несуществующие адреса, дубликаты, временные и одноразовые ящики, спам-ловушки.

**Валидные адреса email** - существующие и правильно написанные адреса, не замеченные в подозрительных действиях. То есть те, которые могут получать рассылки.

Рассылки могут быть отправлены ручным способом, так называемые [ручные рассылки](/broken/pages/-M0qk_GYqVFWIEsD1tcz), а также построены целые схемы, так сказать "умные" рассылки, - [автоворонки](/broken/pages/-M1LeK05OGg0843Eo564), когда учитываются разные варианты развития событий.&#x20;

Авторассылки - это цепочки сообщений, целые сценарии, которые пошагово ведут клиента от момента знакомства с продуктом до конечной цели - его покупки, без вмешательств человека и без учета вариативности.

Рассылки могут быть как разовыми, так и массовыми или групповыми.

В данном разделе поговорим об Email-рассылках.

{% hint style="warning" %}
<mark style="color:red;">**Внимание!**</mark> Для email-рассылок из файла осуществляется проверка адресов на их существование (валидность). Если процент невалидных адресов более 5%, то рассылка отменяется. Это сделано во избежание блокировок и попадания в спам.
{% endhint %}

{% hint style="warning" %}
Если **email-сервер получателя** сообщений **блокирует российские ip,** письма будут отправлены и учтены в лимите исходящих сообщений. Но получатель их не увидит.\
\
К сожалению, повлиять на эту ситуацию не в наших силах.
{% endhint %}
